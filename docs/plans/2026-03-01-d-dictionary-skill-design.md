# `/d` 영한 개발자 사전 스킬 디자인

## 개요

영어 단어/관용구를 입력하면 한국어 뜻과 개발 맥락을 제공하는 경량 사전 스킬.
haiku 서브에이전트를 사용하여 메인 세션 모델 변경 없이 최소 비용으로 동작한다.

## 스킬 정보

- **이름**: `d`
- **위치**: `skills/d/SKILL.md`
- **호출**: `/d {word or phrase}`
- **실행 모델**: haiku (Agent 서브에이전트)

## 사용 예시

```
/d resilient
/d break the ice
/d idempotent
```

## 출력 포맷

```
**resilient** /rɪˈzɪliənt/
회복력 있는, 탄력 있는

💻 개발 맥락:
- "resilient system" → 장애 복구가 빠른 시스템
- "resilient to failures" → 실패에 강한
```

### 출력 구성

1. **첫 줄**: **단어** + 발음기호
2. **둘째 줄**: 한국어 핵심 뜻
3. **개발 맥락 섹션**: 코드/커밋/문서에서 자주 쓰이는 조합과 의미
   - 개발과 무관한 일반 단어일 경우 이 섹션 생략

## 구현 메커니즘

```
사용자 → /d {word} → 스킬 로드 → Agent(model: haiku) 스폰
→ haiku가 번역+개발맥락 생성 → 결과 반환 → 사용자에게 출력
```

### 동작 흐름

1. 사용자 입력에서 단어/구문 추출 (args)
2. Agent 도구를 `model: "haiku"`로 호출
3. haiku 에이전트에게 번역 프롬프트 전달
4. 결과를 그대로 사용자에게 출력

## 설계 결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 모델 | haiku 서브에이전트 | 최소 비용, 메인 세션 불변 |
| 출력 언어 | 한국어 뜻 + 개발 맥락 | 개발자에게 실용적 |
| 입력 범위 | 단어 + 관용구/문장 | 유연한 사용성 |
| BF 연동 | 없음 (독립 유틸리티) | BF 파이프라인과 무관 |
