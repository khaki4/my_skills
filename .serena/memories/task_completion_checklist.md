# Task Completion Checklist

## 스킬 파일 수정/생성 시
1. YAML frontmatter 확인 (`name`, `description` 필수)
2. `description`이 "Use when..." 으로 시작하는지 확인
3. 필수 섹션 포함 여부: Overview, When to Use, Instructions, Output Format
4. CLAUDE.md와 일관성 확인 (워크플로우 시퀀스, 설계 원칙)
5. BF-WORKFLOW-GRAPH.md와 계약 일치 확인

## 일반 변경 시
1. `git status`로 변경 파일 확인
2. 변경이 기존 패턴/컨벤션과 일치하는지 확인
3. CLAUDE.md 업데이트 필요 여부 검토 (새 스킬 추가 시)
