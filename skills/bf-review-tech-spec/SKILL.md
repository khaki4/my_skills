# Review Tech Spec

## Overview

생성된 Tech Spec을 Agent Teams 토론 방식으로 다관점 리뷰한다. 리뷰 결과를 사람(개입 ①)에게 제시하여 승인 여부를 판단하게 한다.

## When to Use

- `/create-tech-spec` 완료 직후 자동 실행
- 사용자가 `/review-tech-spec`을 직접 입력했을 때

## Instructions

1. 최신 Tech Spec 문서를 읽는다.

2. 다음 관점에서 리뷰를 수행한다:
   - **아키텍처 정합성**: 기존 패턴과의 일관성, 모듈 경계 위반 여부
   - **보안**: 인증/인가, 데이터 노출, 입력 검증
   - **성능**: 쿼리 최적화, 캐싱 전략, N+1 문제
   - **테스트 가능성**: AC가 테스트 가능한 형태인지, 엣지 케이스 누락 여부
   - **변경 영향도**: 사이드이펙트, 하위 호환성

3. 각 관점별로 발견사항을 정리한다:
   - 🔴 블로커 (반드시 수정)
   - 🟡 권장 수정
   - 🟢 확인 완료

4. 리뷰 결과를 사용자에게 제시한다.

5. 사용자 판단을 기다린다 (사람 개입 ①):
   - 승인 → `/create-epics-and-stories` 실행 안내
   - 수정 요청 → Tech Spec 수정 후 재리뷰

## Output Format

리뷰 결과를 대화로 출력한다. 별도 파일 저장 없음.
